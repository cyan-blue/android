# 使用的知识点

*	主要界面的布局使用ScrollView 加GirdView来实现
*	ViewPager来实现界面的滑动
*	请求数据使用的是AsyncTask，加okhttp
*	对数据的处理使用的Gson，来建立对象
*	自定义adpter来将数据渲染到界面
*	对主要界面的图片做了缓存（Json后续可以添加），使用的Volley
